####Spinner学习笔记：

#####执行步骤
* 1.　添加一个下拉列表项的list，添加的项就是下拉列表的菜单项
* 2.　为下拉列表定义一个数组适配器（ArrayAdapter），这里用到前面定义的list
* 3.　为适配器设置下拉列表下拉时的菜单样式
* 4.　将适配器添加到下拉列表
* 5.　为下拉列表设置各种事件的响应，这个事件响应菜单被选中
